# mypkgfordata

The goal of this package is to serve as a data warehouse to use in my blog **R & Flowers 4** at [rflowers4.netlify.com](rflowers4.netlify.com)

## Installation

You can install `mypkgDataforblog` from github with:

```R
# install.packages("devtools")
devtools::install_github("perezp44/mypkgDataforblog")
```

## Example

The pacakge only have some data files, then you can load them in R as ususally with:

```R
my_data <- data(some_data)
```
